The Guessing Game

Press play, guess what number will be generated by entering any number between 1 and 100, it will hint you if you guessed a number that's lower or above the generated number. If your guess is correct the game refreshes and you can play again. I used a factory function in this and wrote some unit tests with mocha and chai.

Words Widget

Enter a sentence and it counts letters in each word, words greater than 4 get highlighted in orange and the longest words get highlighted in red, check the checkbox to hide all the shortest words. I used a factory function, handlebars for displaying and I also wrote tests for the factory function using mocha and chai
